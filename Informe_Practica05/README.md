Nombre de la aplicación:
FestiMap Ecuador

Descripción breve del objetivo de la práctica:

La práctica tuvo como finalidad aplicar JavaScript para crear interactividad en la aplicación web FestiMap Ecuador.
Se implementaron validaciones de formularios, manejo de eventos en botones y la simulación de envío de datos.
Además, se integró el almacenamiento local y la manipulación del DOM para actualizar la interfaz dinámicamente.

Conclusiones
Verónica Aguilar
•	Se implementó un sistema de validación de formularios que permitió asegurar la correcta entrada de datos antes de ser almacenados o procesados, fortaleciendo la integridad de la información dentro de la aplicación.
•	Se desarrolló una estructura modular en JavaScript que facilitó la organización del código, permitiendo conectar las funciones del administrador, la vista de detalle y la agenda de usuario de forma coherente y mantenible.
Xavier Anatoa
•	Se aplicó el manejo del DOM para modificar dinámicamente los elementos de la interfaz, logrando que los botones, formularios y listas respondan en tiempo real a las acciones del usuario sin necesidad de recargar la página.
•	Se integró el uso del localStorage para simular la persistencia de datos, almacenando temporalmente los eventos creados y las preferencias del usuario, demostrando cómo se gestiona información local en una aplicación web.
Angelo Conteron
•	Se verificó el funcionamiento de los botones y eventos interactivos, comprobando que las acciones de guardar, editar, eliminar y filtrar datos se ejecuten correctamente y muestren alertas visuales de confirmación o error.
